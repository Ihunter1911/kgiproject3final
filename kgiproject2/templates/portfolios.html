Diabetes Predictor Machine Learning Algorithm

  • 01. Data source

    National Institute of Diabetes and Digestive and Kidney Diseases.

  • 02. Metrics

    Glucose, Blood Pressure, BMI, Age, Skin Thickness, Insulin, and Diabetes Pedigree were identified as the most relevant predictors of positive/negative diagnoses.

  • 03. Dataset

    Binary [Outcome] data point which represents diagnosis of positive/negative.

  • 04. Results

    • Using Random Forest Classifier
    • Accuracy on training set: 80%
    • Accuracy on test set: 75%
    • 70% of the data was trained, and 30% tested (769 Rows x 9 Columns)

Heart Disease Algorithm

    University of Cleveland (1025 Rows x 14 Columns).There is a Binary [Outcome] column which indicates positive/negative diagnoses. Patient metrics include Age, Sex, Cholesterol, etc

    A Machine Learning Model was created and trained, which iterates through each metric and attempts to determine the weights of each individual bias
